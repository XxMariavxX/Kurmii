# Kurmi — Daily Word Guess Game

A simple web-based game inspired by Wordle, where players try to guess a hidden word each day.

## Requirements

- Node.js 22 LTS

## 🎯 About the Project

Kurmi generates a new word every day. The player has a limited number of attempts to guess the correct word.

After each guess, the game provides feedback:

🟩 Correct letter in the correct position

🟨 Correct letter in the wrong position

⬜ Letter not in the word (grey)
